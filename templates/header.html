About

This is the avatar I created in 2007 using the free "Simpsonize Me" app which was featured in Burger King's Simpsons movie promotion. Lihuedon is the user name I adopted when I retired on Kauai and needed a non-work e-mail account. I have used both on various social media sites including Facebook, Twitter, Imgur, BlueSky, etc. It personifies my affinity for the Simpsons fond but unsparing parody of humanity.

This website is my tinker toy. It is hosted on a $10 Raspberry PI Zero located in my home office. It is powered by a Python Flask application running on a Gunicorn Web Service Gateway Interface (WSGI) server.
